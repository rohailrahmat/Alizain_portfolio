# Ali Zain — Professional Video Editor Portfolio

A high-end, world-class professional portfolio built for **Ali Zain**, a Senior Video Editor specializing in social media, commercial, cinematic, and music video content.

## ✨ Key Features
- **Multi-Theme Architecture**: Switch between Cinematic, Purple Creative, and Minimal Pro themes.
- **Dynamic Interactions**: High-performance GSAP animations, scroll reveals, and interactive cursor effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and ultra-wide screens.
- **Expertise Showcase**: Clearly defined sections for services, career experience, and real client work.
- **Modern Tech Stack**: Built using HTML5, CSS3 (Vanilla), and GSAP for animations.

## 🎨 Themes
- **Cinematic**: Futuristic, high-contrast neon mint on rich black.
- **Purple Creative**: Elegant midnight indigo with electric purple accents.
- **Minimal Pro**: Clean, agency-style white background with sophisticated coral tones.

## 🛠️ Built With
- **GSAP**: For complex scroll-triggered animations and UI interactions.
- **Outfit & Inter Fonts**: Professional typography curated for high-authority branding.
- **FontAwesome**: For sleek industry iconography.
